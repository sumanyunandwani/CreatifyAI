# CreatifyAI

A project with separate frontend and backend servers.

## Getting Started

### Prerequisites

- Node.js (for frontend)
- Python 3.x (for backend)
- npm or yarn
- pip

### Running the Frontend

```bash
cd frontend/creatifyAI
npm install
npm start
```

### Running the Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
